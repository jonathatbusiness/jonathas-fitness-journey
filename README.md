# Jonatha's Fitness Journey

Website simples em React + Vite para consulta mobile do plano de treino A/B.

## Objetivo

Organizar um treino full body em dias alternados, com foco em:

- melhora de postura;
- redução de massa gorda;
- definição muscular;
- fortalecimento de costas, core, glúteos e ombros;
- rotina prática para treinos de até 1h.

## Stack

- React
- Vite
- Framer Motion
- Lucide React
- CSS puro

## Como rodar

```bash
npm install
npm run dev
```
